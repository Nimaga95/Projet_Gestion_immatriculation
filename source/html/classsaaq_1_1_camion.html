Paramètres
[in]p_niv: Le numéro de série du camion (ou NIV).
[in]p_immatriculation: Le numéro de plaque d’immatriculation du camion.
[in]p_poids: Le poids du Camion.
[in]p_nbEssieux: Le nombre d'essieux du Camion.
Précondition
(m_poids > 3000) : Le poids du camion doit être supérieur à 3000 kg.
(m_nbEssieux >= 2) : Le nombre d'essieux du camion doit être supérieur ou égale à 2.
(validerImmatriculationCamion (p_immatriculation)) : Le numéro de plaque d’immatriculation du camion doit être dans un format valide pour un camion.
Postcondition
(m_poids == p_poids) : Vérifie que le poids du camion est bien initialisé avec la valeur passée en paramètre.
(m_nbEssieux == p_nbEssieux) : Vérifie que le nombre d'essieux du camion est bien initialisé avec la valeur passée en paramètre.
